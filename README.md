# java-lotto

## 비즈니스 요구 사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.
- 로또는 1개이상 구매해야한다.

## 구현 기능 목록
- [x] 구입 금액 사용자 입력을 받는다.
- [x] 구입 금액에 맞춰 로또를 구입 갯수를 계산한다.
- [x] 구입한 로또의 갯수를 출력한다.
- [x] 구입한 로또의 목록을 생성한다.
- [x] 구입한 로또의 목록을 출력한다.
- [ ] 지난 주 당첨 번호를 입력받는다.
- [ ] 보너스 볼을 입력 받는다.
- [ ] 당첨 결과를 판단한다.
- [ ] 당첨 결과를 출력한다.력
